## 📌 About

**Abhay Kumar Yadav — 3D Portfolio** is a visually immersive and modern web portfolio built to showcase my:

- 👨‍💻 Projects & technical skills  
- 🎨 Creativity through 3D experiences  
- ⚛️ Frontend development expertise  
- 🚀 Passion for modern web technologies  

This portfolio is designed to make a **strong first impression** on recruiters, hiring managers, and collaborators.

---

## ✨ Features

- 🎨 Interactive **3D scenes** using Spline  
- ⚡ High-performance UI built with React + Vite  
- 📱 Fully responsive for all devices  
- 🎬 Smooth animations & transitions  
- 🧩 Clean sections:
  - About Me  
  - Skills  
  - Projects  
  - Contact  

---

## ⚙️ Tech Stack

**Frontend**
- React.js  
- Vite  

**Styling**
- Tailwind CSS  

**Animations**
- Framer Motion  

**3D Integration**
- Spline 3D  

**Deployment**
- Vercel / Netlify
